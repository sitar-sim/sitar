#To install mkdocs (on linux):

Create virtual environment
```
python3 -m venv venv
```

Activate it
```
source venv/bin/activate
```

Install from requirements.txt
```
pip install -r requirements.txt
```

To deactivate later: 
```
deactivate
```

# To build docs
```
mkdocs build
```

# Docs preview:
```
  mkdocs serve
```
Then open the given localhost url in a browser.
